# crud-basics-first
Estudo sobre Java Web

Foi desenvolvida uma API REST, seguindo o framework Spring Boot 2, para realizar as operações CRUD em um banco de dados MySQL.

# Tech Stack
* Java 8
* SQL

# Dependências Utilizadas

* Web
* JPA
* Lombok
* MySQL

      src

        |  ├ controller

        |  ├ dao
        |  |  └ ContactRepository.java
        |  ├ model
        |  |  └ Contact.java
        |  └ App.java
        └ resources
